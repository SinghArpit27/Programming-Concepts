# Programming-Concepts

---

## Here are some questions with test cases on the "if-else" concept, categorized as easy, medium, and hard.

---
## Easy Level
#### These questions and test cases are designed to reinforce basic "if-else" logic.
---

### 1. **Check if a number is positive, negative, or zero**
   - **Question**: Write a program that checks if a given number is positive, negative, or zero.

   - **Test Cases**:
     1. Input: `5` → Output: `Positive`
     2. Input: `-3` → Output: `Negative`
     3. Input: `0` → Output: `Zero`

---

### 2. **Check if a number is even or odd**
   - **Question**: Write a program to check if a given number is even or odd.

   - **Test Cases**:
     1. Input: `4` → Output: `Even`
     2. Input: `7` → Output: `Odd`
     3. Input: `0` → Output: `Even`

---

### 3. **Check the largest of two numbers**
   - **Question**: Write a program that takes two numbers and prints the largest one.

   - **Test Cases**:
     1. Input: `10, 20` → Output: `20`
     2. Input: `15, 5` → Output: `15`
     3. Input: `8, 8` → Output: `Both numbers are equal`

---

### 4. **Check if a number is divisible by 5**
   - **Question**: Write a program to check if a number is divisible by 5.

   - **Test Cases**:
     1. Input: `25` → Output: `Divisible by 5`
     2. Input: `14` → Output: `Not divisible by 5`
     3. Input: `0` → Output: `Divisible by 5`

---

### 5. **Check if a character is a vowel or consonant**
   - **Question**: Write a program to check if a given character is a vowel or a consonant.

   - **Test Cases**:
     1. Input: `'a'` → Output: `Vowel`
     2. Input: `'z'` → Output: `Consonant`
     3. Input: `'E'` → Output: `Vowel`

---

### 6. **Check if a number is between 1 and 100**
   - **Question**: Write a program to check if a number lies between 1 and 100.

   - **Test Cases**:
     1. Input: `50` → Output: `In range`
     2. Input: `150` → Output: `Out of range`
     3. Input: `1` → Output: `In range`

---

### 7. **Check the eligibility to vote**
   - **Question**: Write a program to check if a person is eligible to vote (age should be 18 or more).

   - **Test Cases**:
     1. Input: `18` → Output: `Eligible to vote`
     2. Input: `16` → Output: `Not eligible to vote`
     3. Input: `21` → Output: `Eligible to vote`

---

### 8. **Check if a number is a multiple of 3 and 7**

     4. Input: `10` → Output: `Not a multiple of 3 or 7`

---

### 9. **Check if a year is a leap year**
   - **Question**: Write a program to check if a year is a leap year.

## Easy Level
#### These questions and test cases are designed to reinforce medium-level "if-else" logic.
---

### 1. **Check if a number is within a specific range**
   - **Question**: Write a program to check if a number is between 50 and 150 (inclusive). 

   - **Test Cases**:
     1. Input: `75` → Output: `Within range`
     2. Input: `49` → Output: `Out of range`
     3. Input: `150` → Output: `Within range`
     4. Input: `151` → Output: `Out of range`

---

### 2. **Check if a number is prime**
   - **Question**: Write a program to check if a given number is prime.

   - **Test Cases**:
     1. Input: `2` → Output: `Prime`
     2. Input: `9` → Output: `Not prime`
     3. Input: `13` → Output: `Prime`
     4. Input: `1` → Output: `Not prime`

---

### 3. **Check if a string is a palindrome**
   - **Question**: Write a program to check if a given string is a palindrome (case-insensitive).

   - **Test Cases**:
     1. Input: `"Madam"` → Output: `Palindrome`
     2. Input: `"hello"` → Output: `Not a palindrome`
     3. Input: `"Racecar"` → Output: `Palindrome`
     4. Input: `"Python"` → Output: `Not a palindrome`

---

### 4. **Check if a number is a perfect square**
   - **Question**: Write a program to check if a given number is a perfect square.

   - **Test Cases**:
     1. Input: `16` → Output: `Perfect square`
     2. Input: `20` → Output: `Not a perfect square`
     3. Input: `25` → Output: `Perfect square`
     4. Input: `18` → Output: `Not a perfect square`

---

### 5. **Determine the grade based on score**
   - **Question**: Write a program to determine the grade based on the score: 
     - `A` for score ≥ 90
     - `B` for score between 80 and 89
     - `C` for score between 70 and 79
     - `D` for score between 60 and 69
     - `F` for score < 60

   - **Test Cases**:
     1. Input: `95` → Output: `A`
     2. Input: `85` → Output: `B`
     3. Input: `72` → Output: `C`
     4. Input: `65` → Output: `D`
     5. Input: `55` → Output: `F`

---

### 6. **Determine if a person qualifies for a loan**
   - **Question**: Write a program to check if a person qualifies for a loan based on:
     - Age ≥ 18
     - Monthly income ≥ $2000
     - Credit score ≥ 700

   - **Test Cases**:
     1. Input: `age=25`, `income=2500`, `credit_score=750` → Output: `Qualifies`
     2. Input: `age=30`, `income=1800`, `credit_score=720` → Output: `Does not qualify`
     3. Input: `age=20`, `income=2100`, `credit_score=680` → Output: `Does not qualify`
     4. Input: `age=40`, `income=3000`, `credit_score=720` → Output: `Qualifies`

---

### 7. **Check if a number is divisible by both 3 and 5, or neither**
   - **Question**: Write a program to check if a number is divisible by both 3 and 5, or neither.

   - **Test Cases**:
     1. Input: `15` → Output: `Divisible by both 3 and 5`
     2. Input: `9` → Output: `Divisible by 3 but not by 5`
     3. Input: `10` → Output: `Divisible by 5 but not by 3`
     4. Input: `7` → Output: `Neither`

---

### 8. **Check if a year is a century year**
   - **Question**: Write a program to check if a given year is a century year (ending with `00`).

   - **Test Cases**:
     1. Input: `2000` → Output: `Century year`
     2. Input: `1900` → Output: `Century year`
     3. Input: `2024` → Output: `Not a century year`
     4. Input: `2100` → Output: `Century year`

---

### 9. **Determine the quadrant of a point in a 2D plane**
   - **Question**: Write a program to determine which quadrant a point `(x, y)` is in:
     - Quadrant I: `x > 0` and `y > 0`
     - Quadrant II: `x < 0` and `y > 0`
     - Quadrant III: `x < 0` and `y < 0`
     - Quadrant IV: `x > 0` and `y < 0`
     - Origin if `x == 0` or `y == 0`

   - **Test Cases**:
     1. Input: `(3, 4)` → Output: `Quadrant I`
     2. Input: `(-3, 4)` → Output: `Quadrant II`
     3. Input: `(-3, -4)` → Output: `Quadrant III`
     4. Input: `(3, -4)` → Output: `Quadrant IV`
     5. Input: `(0, 4)` → Output: `On y-axis`
     6. Input: `(3, 0)` → Output: `On x-axis`

---

### 10. **Check if a number is within a specified range excluding boundaries**
   - **Question**: Write a program to check if a number is strictly between 10 and 50 (not inclusive).

   - **Test Cases**:
     1. Input: `30` → Output: `Within range`
     2. Input: `10` → Output: `Out of range`
     3. Input: `50` → Output: `Out of range`
     4. Input: `5` → Output: `Out of range`
    

These questions provide a medium level of complexity, requiring a deeper understanding of the "if-else" concept and logic.
